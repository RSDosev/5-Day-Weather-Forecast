# 5-Day-Weather-Forecast

## Description
Android application which shows weather forecast for the next 5 days. 
It uses [OpenWeatherMap 5 day weather forecast API.](http://openweathermap.org/forecast5)


## How to run
1. Clone project repository
2. Open **Android Studio**
3. Import the project
4. Run on a simulator or real device
